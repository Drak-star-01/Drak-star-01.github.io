1) Pick a data set repository. We will use [https://github.com/openfootball/eng-england](https://github.com/openfootball/eng-england)

2) install sportdb

`gem install sportdb`

Event Keys:
  league
  season
  start_at / begin_at (%Y-%m-%d) date
  end_at / stop_at date
  grounds, stadiums, venues array
  teams array
  fixtures / sources

Goals format
  [Neymar 12']
